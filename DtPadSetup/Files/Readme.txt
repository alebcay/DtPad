1 - INTRODUZIONE
==============================

DtPad è un'applicazione Windows nata per diletto sulla scia di un vecchio editor scritto in Java ai tempi degli studi universitari. Lo scopo è quello di definire un ambiente proprio di editing testuale, ove poter eventualmente sviluppare funzioni nuove o semplicemente ridisegnate in base alle proprie esigenze. In realtà, DtPad si pone come via di mezzo tra la sobrietà di Blocco Note e la complessità di altri programmi, con qualche particolarità che lo rende "unico".

Il suo scopo è dunque quello di assumere il ruolo di sostituto di Blocco Note come strumento rapido da utilizzare, ma al tempo stesso fornito di funzioni tali da permettere una più completa elaborazione di contenuti testuali.

Informativa privacy: DtPad non raccoglie alcuna informazione riguardante uso e utenti. I dati che possono essere storicizzati nella cartella "Logs" riguardano solamente errori e avvisi di programma che sono mostrati anche nell'interfaccia, e in nessun caso essi contengono testo visualizzato nell'area di editing o altri dati privati. Per ulteriori informazioni sulla licenza d'uso fare riferimento al file "License.txt".

2 - REQUISITI
==============================

- Microsoft Windows XP/Vista/7 (32/64 bit)
- Microsoft .NET Framework 4

Nota: Per verificare se si dispone del Framework .NET 4 aprire il "Pannello di controllo" di Windows, cliccare su "Installazione applicazioni" e verificarne la presenza della voce "Microsoft .NET Framework 4". Nel caso in cui non sia presente, è possibile scaricarlo gratuitamente dal sito web di Microsoft [link: www.microsoft.com/downloads/it-it/details.aspx?FamilyID=0a391abd-25c1-4fc0-919f-b21f31ab88b7].


==============================
3 - CARATTERISTICHE
==============================

- Installazione guidata con prima configurazione tramite DtPad Setup
- Utilizzo del registro di Windows solo per salvare l'associazione "Apri con" nel menu contestuale di Esplora risorse, mentre Tutte le altre impostazioni vengono salvate dal programma nella cartella di installazione, con conseguente portabilità (DtPad può anche essere installato su unità disco esterne)
- Interfaccia semplice e chiara con tutte le principali funzioni di un editor testuale
- Supporto multilingua (interfacce disponibili: italiano, inglese - dizionari linguistici disponibili: italiano, inglese, francese, tedesco, spagnolo)
- Gestione di più aree di editing contemporanee
- Gestione e memorizzazione di sessioni di lavoro
- Supporto creazione e lettura da PDF (l'esito di quest'ultima dipende dal tipo di PDF)
- Integrazione con Dropbox (storicizzazione file), Google (ricerca e traduzione) e Wikipedia (ricerca)
- Ricerca funzionale e rapida tramite pannello integrato nell'interfaccia (nessun popup a comparsa)
- Ricerca testo in file multipli a partire da un percorso base
- Editing strutturato del file "hosts" di Windows
- Strumenti di gestione e verifica di file HTML e XML
- Gestione collegamenti ad altri applicativi
- Area di gestione note (persistente) e pannello di annotazione (temporaneo) per appuntarsi informazioni
- Storico delle operazioni di copia/taglia/incolla operate all'interno di una sessione di lavoro, con possibilità di recupero
- Esplora risorse, calcolatrice e calendario integrati
- Aggiornamento rapido tramite DtPad Updater ("?" -> "Controlla aggiornamenti...") per restare aggiornati senza perdere le proprie impostazioni
- Possibilità di ridurre la finestra in tray o di mantenerla sempre in primo piano
- Molto altro ancora...


==============================
4 - INSTALLAZIONE
==============================

DtPad è disponibile in due versioni: setup autoconfigurabile o archivio RAR.

Il setup permette, a differenza dell'archivio, di effettuare una prima semplice impostazione di DtPad e di creare i collegamenti al programma su desktop, menu Start e cartella di "Invia a". Per utilizzarlo, fare doppio clic sull'icona e seguire le indicazioni.

L'archivio consiste invece nei file componenti DtPad comodamente compattati. E' sufficiente aprire il file con un qualsiasi programma di archiviazione e posizionarlo dove si desidera (anche unità disco esterne come chiavette USB, dischi fissi, dischi di rete, ecc...).

5 - DIZIONARI LINGUISTICI
==============================
Per utilizzare la funzionalità relativa a dizionari e sinonimi (disponibile dalla versione 2.6.0.40 in poi) è necessario scaricare i pacchetti che si desidera utilizzare e scompattarne il contenuto sotto la cartella "Languages" che si trova nel percorso dove si è posizionato DtPad (es. tipicamente "C:\Programmi\DtPad\Languages"). Il pacchetto di installazione di DtPad infatti non contiene alcuno dei pacchetti soprastanti per ridurre l'impatto in termini di download.

6 - AGGIORNAMENTI
==============================

Per restare aggiornati senza dover visitare il sito, è possibile effettuare una rapida richiesta direttamente dal menu di "?", cliccando alla voce "Controlla aggiornamenti...".
Alternativamente si può scaricare la versione più recente direttamente dalla pagina di DtPad [link: www.diariotraduttore.com/diariorama/dtpad] e sovrascrivere la propria copia del programma, anche se così facendo le proprie impostazioni andranno perse.


==============================
7 - DUBBI RICORRENTI
==============================

 Domanda: DtPad non parte e al suo posto mi appare un errore di sistema. A cosa è dovuto?
Risposta: DtPad richiede per poter funzionare il .NET Framework 4 di Microsoft. Per verificare se si dispone di esso aprire il "Pannello di controllo", cliccare su "Installazione applicazioni" e verificarne la presenza alla voce "Microsoft .NET Framework 4". Nel caso in cui non sia presente, è possibile scaricalo dal sito web di Microsoft [link: www.microsoft.com/downloads/it-it/details.aspx?FamilyID=0a391abd-25c1-4fc0-919f-b21f31ab88b7].

 Domanda: Si è verificato un errore durante l'utilizzo del programma. Come faccio a segnalarlo di modo che venga corretto in una nuova release?
Risposta: DtPad permette la segnalazione di bug dal menu "?", cliccando alla voce "Segnala errore...". Una volta valorizzati i campi verrà aperto il proprio client di posta e verrà data la possibilità di modificare il testo della segnalazione prima di inoltrarla. Alternativamente si può usare l'apposito modulo di contatto disponibile sul sito dell'autore [link: www.diariotraduttore.com/contatti] specificando i dati relativi all'errore (area interessata, messaggio, azioni effettuate).

 Domanda: Ho un dubbio su una funzionalità di DtPad. Dove posso trovare aiuto?
Risposta: DtPad utilizza il programma DtHelp (sempre a cura dello stesso autore) come sistema di guida offline. Per poterla consultare è sufficiente premere il tasto di scelta rapida F1, oppure cliccare sul link d'aiuto che si trova in ogni finestra dell'applicativo dopo aver cliccato sul bottone "?" che si trova nella barra in alto a destra.
